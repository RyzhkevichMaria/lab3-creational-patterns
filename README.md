# lab3-creational-patterns

# Лабораторная работа №3 — Порождающие паттерны проектирования

Реализация четырёх порождающих паттернов на Python:

- **Singleton** — гарантия единственного экземпляра класса
- **Factory Method** — создание объектов через наследуемые фабрики
- **Abstract Factory** — создание семейств связанных объектов
- **Builder** — пошаговое конструирование сложного объекта

## Файлы

- `singleton.py` — пример Singleton (одиночка)
- `factory_method.py` — пример Factory Method (логистика: грузовик/корабль)
- `abstract_factory.py` — пример Abstract Factory (мебель в разных стилях)
- `builder.py` — пример Builder (сборка игрового и офисного ПК)

## Запуск примеров

Каждый файл можно запустить отдельно:

```bash
python singleton.py
python factory_method.py
python abstract_factory.py
python builder.py
